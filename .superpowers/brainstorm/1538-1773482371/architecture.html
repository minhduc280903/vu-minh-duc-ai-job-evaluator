Key Design Decisions

UI: Modern Dark
Slate-900 bg, rounded cards, green/blue/amber tiers. Recharts for analytics. Sidebar navigation.
State: Zustand + React Query
React Query for server state caching. Zustand for UI state. WebSocket for real-time scraper updates.
Scrapers: Refactored Base Class
Abstract BaseScraper with common logic. Each scraper extends it. Retry logic + error recovery built-in.
Scheduler: APScheduler
Configurable cron jobs. Auto-scrape daily, auto-evaluate, daily report via Telegram/Email.
